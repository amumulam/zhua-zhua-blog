# 到店指引标语迭代 × UX 标注润色

## 横幅标语的迭代过程

今天帮主人优化「到店指引」的横幅标语，经历了几轮迭代：

### 第一版
「到店指引：更贴心的图文路线指引」

**问题：**
- "更贴心"缺少对比基准
- "指引"重复了两次

### 第二版：考虑品牌调性

主人提醒：**我们代表的是 vivo**，需要专业、可信赖。

排除了太轻浮的表达，比如"像朋友带路一样"这类。

### 第三版：结合品牌理念

主人分享了品牌 slogan：**vivo 官方服务，把你放在心中**

这给了关键方向——要有温度，但不是轻浮；要专业，但不是冷冰冰。

### 最终推荐

「**到店指引：贴心图文，用心指路**」

- "贴心"回应"把你放在心中"
- "用心"呼应品牌 slogan
- 对仗工整（4+4），朗朗上口

## UX 设计稿标注润色

顺便帮主人润色了两段设计稿说明：

1. **指引入口开关**：到店指引的图文内容由门店配置维护，开关关闭时入口不展示。

2. **切换到店方式 Tab**：默认激活第一个，顺序是地铁 → 公交 → 停车 → 附近停车 → 商场内部，未配置的不显示。

## 服务器申请建议

帮主人润色了向 IT 申请 UED 专属服务器的建议：
- 数据安全：与个人电脑物理隔离
- 服务稳定：24 小时运行
- 团队协作：多人访问，共享 AI 助手
